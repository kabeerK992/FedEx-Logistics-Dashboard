# 🚀 FedEx Logistics Analysis Dashboard

![Dashboard Preview](Fedex_Logistics_Dashboard_.png)

## 📊 Project Overview
An interactive Excel dashboard built to analyze 
FedEx Supply Chain logistics data from the 
SCMS Delivery History Dataset (2006–2015).

## 📁 Files
| File | Description |
|------|-------------|
| `SCMS_Delivery_History_Dataset.csv` | Raw data |
| `FedEx_Dashboard.xlsx` | Excel Dashboard file |
| `Fedex_Logistics_Dashboard_.png` | Dashboard screenshot |

## 📈 Dashboard Features
- ✅ 5 KPI Cards (Shipments, Freight, Value, On-Time Rate, Delay)
- ✅ 6 Interactive Charts
- ✅ 5 Slicers (Month, Country, Mode, Product, Fulfill Via)
- ✅ FedEx branded header
- ✅ Pivot Tables with dynamic filters

## 🔢 Key Metrics
| Metric | Value |
|--------|-------|
| Total Shipments | 7,900 |
